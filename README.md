# Student Grade Tracker

A Java Swing-based Student Grade Tracker developed as part of the CodeAlpha Internship.

## Features

- Add Student Records
- Update Student Details
- Delete Student Records
- Search by Roll Number
- Automatic Grade Calculation
- Average Marks Calculation
- Highest and Lowest Score Tracking
- Pass Percentage Calculation
- Top Performer Detection
- CSV Report Export
- Persistent Data Storage using Serialization

## Technologies Used

- Java
- Swing
- JTable
- ArrayList
- File Handling
- OOP Concepts
