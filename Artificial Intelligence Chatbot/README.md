# Artificial Intelligence Chatbot

## CodeAlpha Internship - Task 2

### Description

A Java-based AI Chatbot developed using Java Swing. The chatbot provides interactive communication through a graphical user interface and responds to frequently asked questions using rule-based logic.

### Features

* Java Swing GUI
* Interactive chatbot interface
* FAQ-based responses
* Date and time responses
* Motivational quotes
* Chat history storage
* Dark-themed user interface

### Technologies Used

* Java
* Java Swing
* HashMap
* File Handling

### Project Structure

```
Artificial Intelligence Chatbot
│
├── src
│   ├── ChatBotGUI.java
│   ├── ChatBotLogic.java
│   └── ChatHistory.java
│
└── README.md
```

### How to Run

Compile:

```
javac *.java
```

Run:

```
java ChatBotGUI
```
